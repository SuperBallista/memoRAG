================================================================================
                    최종 배포 패키지 만들기
================================================================================

빌드가 완료되면 사용자들에게 전달할 배포 패키지를 만듭니다.


■ 배포 폴더 구성

memoRAG_v0.1.0_windows/
├── memoRAG.exe                      ← 실행 파일
├── README.txt                      ← 필수! 여기부터 읽기
├── 빠른_시작_가이드.txt            ← 5분 시작
├── 사용자_가이드.txt               ← 자세한 사용법
├── 부서_공유_가이드.txt            ← 팀 활용법
├── 명령어_치트시트.txt             ← 복사용
└── 샘플문서/                       ← 연습용
    ├── sample1.txt
    ├── sample2.txt
    └── sample3.txt


■ 만드는 방법

Step 1. 빌드 완료 확인
────────────────────
dist 폴더에 memoRAG.exe가 있는지 확인


Step 2. 배포 폴더 생성
────────────────────
새 폴더 만들기: memoRAG_v0.1.0_windows


Step 3. 파일 복사
────────────────────
다음 파일들을 배포 폴더로 복사:

✓ dist\memoRAG.exe                    → memoRAG.exe
✓ docs\README_사용자용.txt          → README.txt
✓ docs\빠른_시작_가이드.txt         → 빠른_시작_가이드.txt
✓ docs\사용자_가이드.txt            → 사용자_가이드.txt
✓ docs\부서_공유_가이드.txt         → 부서_공유_가이드.txt
✓ docs\명령어_치트시트.txt          → 명령어_치트시트.txt
✓ test_documents\                   → 샘플문서\


Step 4. 압축하기
────────────────────
memoRAG_v0.1.0_windows 폴더를 우클릭
→ "보내기" → "압축(zip) 폴더"

결과: memoRAG_v0.1.0_windows.zip


Step 5. 공유하기
────────────────────
zip 파일을:
- 학교 홈페이지 업로드
- 구글 드라이브 공유
- 이메일 전송
- USB 복사


■ 배포 시 안내 메시지 (복사해서 사용)

────────────────────────────────────────────────────
[안내] memoRAG 문서 검색 프로그램 배포

교사 여러분,

문서를 쉽게 찾을 수 있는 프로그램을 공유드립니다.

▶ 다운로드
첨부된 memoRAG_v0.1.0_windows.zip 파일

▶ 설치 방법
1. zip 파일 압축 해제
2. README.txt 먼저 읽기
3. memoRAG.exe 실행

▶ 특징
- 한글, 워드, PDF 등 모든 문서 검색
- "체육대회 준비물" 같은 자연어로 검색
- 완전 무료, 보안 안전

▶ 문의
문제가 있으시면 말씀해주세요!
