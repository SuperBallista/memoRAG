■ memoRAG란?

"메모(memo)" + "RAG(검색증강생성)"의 합성어입니다.

여러분의 모든 업무 메모, 문서, 자료를
저장하기만 하면 나중에 질문으로 쉽게 찾을 수 있는
스마트 문서 검색 시스템입니다.


■ 누구를 위한 프로그램인가요?

👨‍🏫 교사
"지난달 체육대회 준비물이 뭐였지?"
"학부모 상담 일정 언제라고 공지했더라?"

💼 직장인
"지난 분기 회의에서 뭐라고 결정했지?"
"신제품 기획서 어디 저장했더라?"

📚 연구자
"이 논문 어디서 봤더라?"
"참고문헌 어디 있지?"

📋 프리랜서/1인 기업
"작년 클라이언트 요구사항이 뭐였지?"
"견적서 어디 갔지?"


■ 왜 memoRAG인가요?

기존 방식:
❌ 폴더를 일일이 뒤지기
❌ "파일명이 뭐였더라..." 고민
❌ 여러 곳에 흩어진 문서들
❌ 시간 낭비

memoRAG 사용:
✓ 평소 말하듯이 질문
✓ 1초 안에 관련 문서 찾기
✓ 어느 폴더에 있든 한 번에 검색
✓ 시간 절약!


■ 핵심 철학

1. "저장하고 잊어버려도 OK"
   - 어디 저장했는지 기억 안 나도 찾아줌
   - 파일명 몰라도 내용으로 검색

2. "보안 제일"
   - 모든 처리가 내 컴퓨터에서만
   - 외부 서버 업로드 전혀 없음
   - 언제든 삭제 가능

3. "간편함이 최고"
   - Python 설치 불필요
   - 복잡한 설정 없음
   - 명령어 3개만 알면 됨

4. "팀과 함께"
   - 내 문서만 검색할 수도
   - 팀 문서를 공유할 수도
   - 선택은 자유!


■ 실제 활용 사례

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Case 1: 초등학교 김선생님

업무 현황:
- 매일 5~10개 공문, 안내문 수신
- 학급 운영 자료 수백 개
- "이거 언제 받았더라?" 자주 발생

memoRAG 도입 후:
✓ "현장학습 준비물" 검색 → 즉시 찾음
✓ "학부모 상담" 검색 → 관련 안내문 모두 나열
✓ 문서 찾는 시간 1일 30분 → 5분으로 단축

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Case 2: 스타트업 기획팀 박대리

업무 현황:
- 회의록, 기획서, 보고서 산더미
- 여러 프로젝트 동시 진행
- "그때 뭐라고 했더라?" 기억 안 남

memoRAG 도입 후:
✓ "신제품 출시 일정" 검색 → 회의록에서 발견
✓ "경쟁사 분석" 검색 → 관련 보고서 찾음
✓ 자료 찾는 시간 주 3시간 → 30분으로 단축

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Case 3: 대학원생 이씨

연구 현황:
- 논문 PDF 수백 개
- 연구노트, 실험 결과 자료
- "이 내용 어느 논문이었지?" 헤맴

memoRAG 도입 후:
✓ "강화학습 알고리즘" 검색 → 관련 논문 목록
✓ "실험 2번 결과" 검색 → 노트 발견
✓ 문헌 찾는 시간 주 5시간 → 1시간으로 단축
