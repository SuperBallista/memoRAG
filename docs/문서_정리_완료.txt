■ 주요 개선 사항

✅ 문서를 용도별로 분리
   - dev/ : 개발자용 (기술 문서)
   - user/ : 사용자용 (사용 설명서)

✅ 각 폴더에 README 추가
   - 해당 폴더의 가이드 역할

✅ 중복 파일 제거
   - 깔끔한 구조

✅ 향후 계획 업데이트
   - GUI 최우선 (UX/UI 디자인 포함)
   - 클라우드 연동 (Notion, Google, OneNote)
   - HWP 지원 제외 (우선순위 낮음)
