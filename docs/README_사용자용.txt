                            memoRAG란?

여러분의 컴퓨터에 저장된 수많은 문서들...
필요할 때 찾기 힘드셨죠?

"저번에 받은 체육대회 안내문 어디 갔지?"
"학부모 공개수업이 언제라고 했더라?"

이제 memoRAG에게 물어보세요!
평소처럼 질문하면 관련 문서를 바로 찾아줍니다.


■ 주요 기능

✓ 자연어 검색
  "체육대회 준비물 뭐야?" 이렇게 평소 말하듯이 검색

✓ 다양한 파일 지원
  한글(.hwpx), 워드(.docx), PDF, 엑셀, 파워포인트 모두 가능

✓ 빠른 검색
  1초 안에 수천 개 문서에서 필요한 내용 찾기

✓ 완벽한 보안
  모든 처리가 내 컴퓨터에서만! 외부 유출 걱정 없음


■ 누가 사용하면 좋을까요?

👨‍🏫 교사
- 공문, 안내문, 업무 문서 빠르게 검색
- 학년부/교무부 문서 공유

💼 직장인
- 업무 자료, 회의록, 보고서 관리
- 팀 문서 검색

📚 연구자/학생
- 논문, 자료 검색
- 참고문헌 찾기


■ 시작하기

상세한 사용법은 다음 파일을 참고하세요:

1. 빠른_시작_가이드.txt ← 5분 만에 시작!
2. 사용자_가이드.txt ← 자세한 설명


■ 필요 사항

- Windows 10 이상
- 최소 2GB 여유 공간
- 인터넷 연결 (최초 1회)
